# What is Keylogger?
<ul>
  <li>Key logger is a software or tool either a software or hardware used to record keystrokes made on computer or mobile devices. Its primary purpose is to capture every sensitive information such as passwords, credit card numbers, personal messages etc... typed by user without their knowledge.
</li>
  <li>Keylogger send these recorded keystroke logs to the attacker either via internet or store it locally to be accessed later.</li>
</ul>

<h2>How does keylogger get into the system?]</h2>
<ul>
  <li>Keylogger typically enters to the target system via email phishing, malicious website, software download containing hidden malware.</li>
  <li>Attackers may send keylogger via malicious USB drives or physically install hardware keyloggers devices, if they have direct access to the computer.</li>
</ul>

  <li>The keylogger records each keystroke and stores it in a hidden log file locally or prepares to send it.</li>
  <li>Periodically, the logged keystroke data is automatically sent remotely to the attackerâ€™s server or email.</li>
  <li>The attacker accesses the captured data remotely to extract sensitive information like passwords or credit card numbers.</li>

</ol>
