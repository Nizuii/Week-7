# What is Keylogger?
<ul>
  <li>Key logger is a software or tool either a software or hardware used to record keystrokes made on computer or mobile devices. Its primary purpose is to capture every sensitive information such as passwords, credit card numbers, personal messages etc... typed by user without their knowledge.
</li>
  <li>Keylogger send these recorded keystroke logs to the attacker either via internet or store it locally to be accessed later.</li>
</ul>
