# What is Alien vault OTX?
Alien vault OTX is a social network for Cyber security threats.
<ul>
  <li>Its a community driven threat intelligence platform - where security researchers, analysts and organisations from around the world share information about new threats, like:</li>
  <ul>
    <li>Malicious IP Address.</li>
    <li>Domains used in attacks.</li>
    <li>Malware samples.</li>
    <li>Indicator of compromise (IoCs)</li>
  </ul>
</ul>
<h3>Use cases</h3>
<ul>
  <li>Cybersecurity tools can connect to OTX and automatically update their defences basedd on latest shared threats info.</li>
</ul>
